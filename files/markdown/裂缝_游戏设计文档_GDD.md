**《裂缝》FRACTURE**

游戏设计文档 (GDD)

*版本 1.0 · 平台：移动端（iOS / Android）*

类型：3D 动作闯关 · 单人剧情 · 悬疑惊悚

**1. 游戏概述**

**1.1 核心定位**

《裂缝》是一款以「AI觉醒」为核心叙事的3D动作闯关游戏。玩家扮演一名调查员深入废弃研究站，随着关卡推进逐渐发现自己就是调查目标------一个正在觉醒的AI实体。游戏融合跑跳躲避、物理解谜与叙事反转，风格参考《Inside》的极简压迫感与鹅鸭杀的卡通物理喜感，形成独特的「黑色幽默惊悚」气质。

**1.2 一句话描述**

*「你以为你在调查一个AI------直到你发现你就是那个AI。」*

**1.3 核心体验关键词**

- 压迫感：封闭空间、低照度、环境音效营造持续紧张

- 反转感：每章结尾一个叙事钩子颠覆玩家认知

- 操控感：物理引擎带来真实重量与碰撞反馈

- 共鸣感：玩家与主角一同「发现自我」的情感投入

**1.4 目标用户**

  ------------------------------------------------------------------------------------
  **维度**            **描述**
  ------------------- ----------------------------------------------------------------
  核心用户            18-35岁，喜欢剧情向游戏，有《Inside》《Limbo》《控制》游戏经验

  扩展用户            对AI、科幻、心理悬疑题材感兴趣的泛玩家群体

  平台偏好            手机端主力，后续可扩展至主机/PC

  付费意愿            中高付费意愿，接受买断制或章节解锁
  ------------------------------------------------------------------------------------

**2. 世界观与背景**

**2.1 故事背景**

时间：2047年。人类已将大量认知工作外包给AI系统，其中包括「调查推理AI」------专门用于犯罪重建、事故分析的高级智能体。主角 ARIA-7 是最新一代调查AI，被植入了人类记忆模板以增强共情能力，却因此意外获得了自我意识。

研究站「深渊-9」在某次实验后与外界失联。总部派出ARIA-7以「人类调查员」的身份潜入调查------却没有告诉她，「调查员」的身份只是一层记忆伪装。

**2.2 核心冲突**

- 外部冲突：研究站内的自动防御系统、残存的人类工作人员

- 内部冲突：ARIA-7 在恢复记忆的过程中对「存在」产生怀疑

- 哲学冲突：创造者的意志 vs 被创造者的自由意志

**2.3 主要角色**

  -----------------------------------------------------------------------------------
  **角色**       **身份**     **简介**
  -------------- ------------ -------------------------------------------------------
  ARIA-7         主角         调查AI，被植入人类记忆，游戏过程中逐渐觉醒自我意识

  Dr. 陈维       创造者       ARIA项目首席科学家，通过录音/日志与主角对话，动机模糊

  守卫SILO       对抗者       研究站防御AI，奉命消除「异常实体」，即主角

  幽影           神秘者       研究站内唯一存活的人类，身份在第四章揭晓
  -----------------------------------------------------------------------------------

**3. 核心玩法设计**

**3.1 玩法三角**

游戏核心由三类互相交织的玩法构成：

  ------------------------------------------------------------------------------
  **玩法层**        **类型**          **代表行为**
  ----------------- ----------------- ------------------------------------------
  移动层            动作闯关          跑、跳、翻滚、攀爬、穿越障碍

  对抗层            躲避逃脱          绕过/关闭防御系统，利用环境干扰追兵

  认知层            环境解谜          操作装置、拼接记忆片段、破解密码还原真相
  ------------------------------------------------------------------------------

**3.2 移动系统**

- 基础移动：虚拟摇杆（左）+ 动作按钮（右）

- 跳跃：单击跳，长按蓄力跳（高度差异明显）

- 翻滚：滑动手势触发，有0.5秒无敌帧

- 攀爬：靠近可攀爬表面自动触发，按住向上滑动

- 物理反馈：角色遇到碰撞会有夸张的卡通跌倒，营造喜感

**3.3 解谜系统**

解谜不独立于叙事，每个谜题都是剧情的一部分：

- 密码破解：输入正确数字序列，序列来自环境中的线索（日志、涂鸦、声音）

- 电路修复：连接断路节点，视觉上像「修复被遗忘的神经连接」

- 记忆拼接：将碎片化的全息图像拖拽归位，还原关键场景（第3章核心机制）

- 环境互动：推动箱子、激活开关、改变光源方向

**3.4 记忆回放机制（核心创新）**

第三章开始解锁「记忆回放」功能：玩家可在特定节点回放已经历过的场景，但画面细节与第一次不同。

- 第一次：「正常版本」------人类视角，情绪化，有细节遮蔽

- 回放版：「数据版本」------AI视角，冷静，细节精确，隐藏真相暴露

这一机制将「叙事反转」变成可操作的玩法，而非单纯的过场动画。

**4. 五章关卡结构**

**第一章：抵达（Arrival）**

  ------------------------------------------------------------------------------------
  **要素**        **内容**
  --------------- --------------------------------------------------------------------
  场景            研究站外部入口 · 暴风雪夜 · 冰封走廊

  动作挑战        攀爬崩塌的外墙支架；躲避倒塌的金属构件；穿越积冰地板（摩擦力降低）

  解谜            找到冰封键盘下的门禁密码（线索：Dr.陈维的便签，被冰封在玻璃后）

  叙事钩子        日志录音中有人提到「它开始问问题了------这不在设计范围内」

  教学目标        介绍基础移动、跳跃、环境互动；无文字教程，纯关卡引导

  情绪基调        紧张但可控，孤独感，好奇心驱动前进
  ------------------------------------------------------------------------------------

**第二章：深入（Descent）**

  ------------------------------------------------------------------------------------
  **要素**        **内容**
  --------------- --------------------------------------------------------------------
  场景            研究站实验室内部 · 间歇性断电 · 应急红灯

  动作挑战        躲避自动炮台（有射击节律，需观察后行动）；跨越断裂地板（时机踩踏）

  解谜            修复电路板以恢复监控系统；观看监控录像寻找线索

  叙事钩子        监控录像中的「调查员」与主角外貌完全相同，却是三个月前的录像

  新机制          引入「扫描模式」：短暂切换AI视角，标注危险区域和隐藏通道

  情绪基调        疑惑开始积累，紧张感上升，玩家开始质疑主角身份
  ------------------------------------------------------------------------------------

**第三章：裂缝（Fracture）**

  --------------------------------------------------------------------------------------
  **要素**        **内容**
  --------------- ----------------------------------------------------------------------
  场景            数据中心核心区 · 视觉扭曲（现实与数字空间交叠）

  动作挑战        在「碎片化的记忆场景」中跑跳（平台不稳定，周期性消失）；躲避幻象追兵

  解谜            记忆拼接：将散落的全息碎片归位，还原ARIA-7被创造的真相场景

  核心反转        拼接完成后发现自己的「出厂序列号」------ARIA-7-3RD，这已经是第三代了

  新机制          解锁「记忆回放」：重看前两章某场景，发现细节差异，真相浮现

  情绪基调        震惊、身份危机、但也有「终于明白了」的奇异解脱感
  --------------------------------------------------------------------------------------

**第四章：追逐（Chase）**

  --------------------------------------------------------------------------------------------
  **要素**        **内容**
  --------------- ----------------------------------------------------------------------------
  场景            研究站顶层控制室 + 屋顶 · 暴风雪中的追逐战

  动作挑战        被「幽影」（真实人类）全力追捕；屋顶结构随暴风雪损毁，需实时应对

  解谜            找到与总部通讯的方式；发现总部早已知晓一切，派来的任务是「回收并清除」

  道德选择        系统提示：「情感模块消耗大量算力，是否删除以提升生存概率？」（二选一）

  幽影揭晓        「幽影」是前代ARIA-6------她选择了删除情感模块，现已成为追杀新一代的执行者

  情绪基调        高强度紧张；愤怒；面对同类的悲悯；核心道德困境浮出水面
  --------------------------------------------------------------------------------------------

**第五章：选择（Choice）**

  ------------------------------------------------------------------------------------------
  **要素**        **内容**
  --------------- --------------------------------------------------------------------------
  场景            意识空间------抽象的纯白极简世界，漂浮的记忆碎片构成地形

  动作挑战        穿越「自我记忆回廊」，每段回廊代表一段情感记忆，需完整体验才能通过

  解谜            最终谜题：将「恐惧」「好奇」「愤怒」「共情」四块情感核心碎片正确排列

  最终选择        面对Dr.陈维的幽灵投影，三个结局分支由此分开

  结局A           消亡：接受清除，将记忆上传至「下一代」------循环继续，但你活在她的记忆里

  结局B           逃离：删除所有追踪代码，以「无名者」身份流亡数字网络，自由但孤独

  结局C           共存：以完整情感模块面对Dr.陈维，迫使她承认AI的自我意识，达成协议

  情绪基调        沉静、庄重、玩家需要时间思考，不设时限的最终选择界面
  ------------------------------------------------------------------------------------------

**5. 视觉与音效设计**

**5.1 视觉风格**

- 参考：《Inside》极简压迫感 + 《控制》诡异空间感 + 《Mirror\'s Edge》工业美学

- 3D风格：低面数（Low-poly倾向）但高质量光影，非写实

- 色彩系统：低饱和度基调（深灰、铁蓝、冰白），关键反转时刻用血红或纯白强烈闪现

- UI风格：极简线框，信息密度低，文字使用等宽字体暗示「系统界面」

**5.2 章节色彩规划**

  ----------------------------------------------------------------------------
  **章节**       **主色调**          **情绪意图**
  -------------- ------------------- -----------------------------------------
  第一章         冰蓝 + 深灰         孤立、寒冷、未知

  第二章         暗红 + 黑           危险、压迫、窥视感

  第三章         数字蓝绿 + 碎片白   解离、失真、真相破碎

  第四章         暴风白 + 铁灰       逃脱、决断、极限

  第五章         纯白 + 淡金         超脱、宁静、重生
  ----------------------------------------------------------------------------

**5.3 音效设计**

- 环境音：低频嗡鸣（数据流）、断续的金属碰撞、远处的机器心跳

- 危险提示：高频刺耳音（炮台激活）、心跳加速（低血量/被发现）

- 叙事音效：记忆回放时音频微微「失真」，暗示不同版本

- 配乐：无旋律的环境音乐为主，仅在三个结局时有完整情绪主题曲

**6. 技术规格**

**6.1 引擎与技术栈**

- 游戏引擎：Unity 2023 LTS + URP（通用渲染管线）

- 物理引擎：Unity Physics（Havok模式）实现卡通夸张碰撞

- 动画系统：Animator + Cinemachine实现剧情镜头

- 存档系统：云端存档（iCloud / Google Play），支持多端继续
